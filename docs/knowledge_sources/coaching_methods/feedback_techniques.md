# フィードバック技法

## 概要
効果的なフィードバックは選手の成長を促進する重要な要素です。

## 基本原則

### 1. 建設的フィードバック
- 具体的で明確な内容
- 改善可能な点に焦点
- ポジティブな要素も含める

### 2. タイミング
- 適切なタイミングでの提供
- 感情的な状態を考慮
- 継続的なフォローアップ

### 3. 個別化
- 個人の特性に合わせたアプローチ
- 理解度に応じた説明
- 段階的な改善提案

## 実践技法

### サンドイッチ法
1. ポジティブな点を指摘
2. 改善点を具体的に説明
3. 励ましと期待を表現

### 質問型フィードバック
- 選手自身の気づきを促す
- 自己評価能力の向上
- 主体的な改善意欲の喚起

### 目標指向フィードバック
- 明確な目標設定
- 進捗の可視化
- 達成感の提供

## 注意点
- 感情的な批判を避ける
- 比較による評価を控える
- 長期的な視点を保つ
